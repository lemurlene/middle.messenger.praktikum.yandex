# Messenger
Учебный проект мессенджера на **TypeScript + Vite + Handlebars**.  
В первом спринте реализованы статические страницы по макету и базовая навигация между ними.


## Страницы
- **/** — авторизация  
- **/sign-up** — регистрация  
- **/messenger** — список чатов и заглушка переписки (поле `message`)  
- **/settings** — профиль пользователя  
- **/404** — страница «Не найдено»  
- **/500** — страница ошибки сервера

## Технологии
- TypeScript
- Vite
- Handlebars
- SCSS (Sass)
